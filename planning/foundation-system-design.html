Durable Data Flow

Postgres is the local-first system of record. pgvector stores retrieval memory in the same operational database.

Recorded State

Runs, conversation turns, agent messages, events, sources, claims, feedback, artifacts, and agent memories are stored as durable records. LangGraph checkpoints sit beside app state instead of replacing it.
