# Source Inventory

This inventory separates official/open sources from local user-provided material. Status values:

- `eligible`: official/public/open source usable now for notes and design synthesis.
- `user_provided_local`: local material the user explicitly provided or pointed Codex at; use compact original notes and design implications, not raw text dumps.
- `official_or_open_local`: local copy appears to correspond to an official/open source; use normally after keeping provenance in the note.
- `blocked`: do not use unless replaced or reverified.

The local book folder has a dedicated corpus note: [[local-books-corpus]].
