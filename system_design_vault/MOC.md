# Agent Studio System Design MOC

## Current Focus

Design a local-first, realtime, long-running multi-agent content studio with:

- durable Postgres + pgvector state,
- LangGraph-style checkpointed orchestration,
- A2A-style agent cards and handoffs,
- provider-pluggable realtime voice,
- OpenRouter `deepseek/deepseek-v4-flash` live dialogue reasoning over LiveKit with Kokoro spoken output,
- retrieval, reranking, graph traversal, and source-ledger precision/recall gates,
- human feedback gates and revision loops,
- Obsidian-first planning memory.
