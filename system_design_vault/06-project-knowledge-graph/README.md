# 06 — Agent Studio Project Knowledge Graph

This directory contains a compact knowledge graph of the Agent Studio project so LLMs can quickly understand what's being built, what's done, what's blocked, and which vault files hold the relevant details — without loading the full vault context.

## How to use this

- Start with `agent-studio-project-kg.md` — a three-part map: System Architecture, Workstream Status, and File Index.
- Each section links to the canonical Obsidian notes in this vault (`system_design_vault/`) and the project planning vault (`social_media_optimiser/`).
- The Project State section at the bottom shows the current completion status per workstream.
